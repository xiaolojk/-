# 卡牌对决游戏

一个类卡片游戏，还在建设中。

## 项目结构

```
/workspace/
├── config.iyu          # 全局配置文件（API地址、颜色主题等）
├── api.myu             # 网络请求模块（注册、登录、用户信息管理）
├── login.iyu           # 登录界面逻辑
├── login.xml           # 登录界面布局
├── register.iyu        # 注册界面逻辑
├── register.xml        # 注册界面布局
├── main.iyu            # 主界面逻辑
├── main.xml            # 主界面布局
└── README.md           # 项目说明
```

## 技术栈

- **开发语言**: 裕语言 (iyu)
- **运行平台**: Android 8.0+
- **UI设计**: Material Design 风格，深色主题

## 功能特性

### 登录界面
- 用户名输入框
- 密码输入框（隐藏明文）
- 登录按钮（含加载状态）
- 跳转注册链接
- 用户协议与隐私政策

### 注册界面
- 用户名输入（至少3个字符）
- 邮箱输入（格式验证）
- 密码输入（至少6个字符）
- 确认密码（一致性验证）
- 返回按钮
- 用户协议与隐私政策

### 网络请求
- 支持 POST 请求
- JSON 数据格式
- 异步线程请求
- 用户信息本地存储

## 使用说明

1. 将项目文件导入 iApp 开发环境
2. 修改 `config.iyu` 中的 API 地址为实际服务器地址
3. 构建并运行项目
